PLACEHOLDER IMAGE GENERATORS (Temporary Use)
================================================================================

If you want to use placeholder images temporarily:

Via Placeholder:
- Format: https://via.placeholder.com/WIDTHxHEIGHT
- Example: https://via.placeholder.com/1920x720

Placehold.co:
- Format: https://placehold.co/WIDTHxHEIGHT
- Example: https://placehold.co/373x469

Lorem Picsum (Random Photos):
- Format: https://picsum.photos/WIDTH/HEIGHT
- Example: https://picsum.photos/1920/720

================================================================================
STEP-BY-STEP REPLACEMENT PROCESS
================================================================================

1. DOWNLOAD YOUR IMAGES
   - Go to stock photo website
   - Search for relevant terms
   - Download images (usually free license)

2. RESIZE IF NEEDED
   - Use free tools like:
     * GIMP (free Photoshop alternative)
     * Online resizers: iloveimg.com/resize-image
     * Preview app on Mac

3. OPTIMIZE FILE SIZE
   - Use TinyPNG.com or TinyJPG.com
   - Reduces file size without losing quality
   - Faster website loading

4. RENAME FILES
   - Keep same names as current images OR
   - Update HTML files to reference new names

5. REPLACE FILES
   - Delete old images
   - Copy new images to same location
   - Refresh browser to see changes
